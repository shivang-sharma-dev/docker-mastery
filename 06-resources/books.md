# Books

| Book | Author | Level | Notes |
|---|---|---|---|
| Docker Deep Dive | Nigel Poulton | Beginner | Short, clear, practical — best starter book |
| Docker in Practice | Ian Miell & Aidan Hobson Sayers | Intermediate | 101 real-world techniques |
| Docker: Up & Running | Sean Kane & Karl Matthias | Intermediate | O'Reilly, comprehensive |
| Container Security | Liz Rice | Advanced | Deep dive into the Linux security model |

Start with **Docker Deep Dive** — it is short (200 pages), opinionated,
and gets you productive fast.
