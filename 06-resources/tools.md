# Tools

### Essential
| Tool | What it does | Install |
|---|---|---|
| Docker Desktop | GUI + Docker engine for Mac/Windows | https://docs.docker.com/get-docker/ |
| Docker Compose | Multi-container apps | Included with Docker Desktop |
| Lazydocker | Terminal UI for Docker | https://github.com/jesseduffield/lazydocker |
| Dive | Explore image layers and find wasted space | https://github.com/wagoodman/dive |

### Security Scanning
| Tool | What it does | Install |
|---|---|---|
| Trivy | Scan images for CVEs (open source, fast) | https://aquasecurity.github.io/trivy |
| Docker Scout | Built into Docker CLI | `docker scout cves myimage` |
| Hadolint | Lint your Dockerfiles | https://github.com/hadolint/hadolint |

### Development
| Tool | What it does |
|---|---|
| ctop | Top-like interface for containers |
| Portainer | Web UI for managing Docker |
| Watchtower | Automatically update running containers when new images push |
